# Keno Game

A JavaFX-based implementation of the classic Keno lottery game. Players select numbers, place bets, and watch the draw to see if they win.

## Files

- `Keno.java` – Core game logic and player management
- `GameScene.java` – Main game UI and interactions
- `WelcomeScene.java` – Start screen for the application
- `BetCard.java` – Handles betting mechanics and card display
- `JavaFXTemplate.java` – Application entry point (launches welcome scene)

## Features

- Select Keno numbers (typically 1–10 spots)
- Place and manage bets
- Random number drawing
- Win/loss calculation based on matches
- Clean JavaFX user interface

## Requirements

- JavaFX SDK
- Java 11 or later
